# 🌱 BeejMitra - Smart Seed Distribution Platform

## Project Overview

BeejMitra is a role-based Seed Distribution and Demand Forecasting Platform developed for agricultural supply chain management.

The platform enables distributors to place seed orders while helping administrators forecast demand, identify risks, estimate profits, and make production decisions using analytics.

---

## Key Features

### Admin Portal
- Dashboard Analytics
- Demand Forecasting
- Production Recommendations
- Profit Estimation
- Risk Analysis
- Order Management

### Distributor Portal
- Product Marketplace

- Shopping Cart

### AI Assistant
