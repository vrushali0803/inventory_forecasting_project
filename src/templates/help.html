Help & Support

We're here to help you use BeejMitra efficiently

🛒 How to place an order?
Browse products → Add to Cart → Open Cart → Checkout → Complete Payment.
🔒 How to change password?
Go to Profile → Security Settings → Enter New Password → Save.
📦 Where can I see previous orders?
Open Profile → Order History to view all your orders.
❌ How to logout?
Open Profile Menu and click Logout.
